# 统一命名与用语表

用于页面标题、导航、按钮、模块名，避免同一概念多种说法。与需求文档中的「测评」「课程」及用户向表述一致。

## 与 i18n 字典的关系

- **单一事实来源（中文）**：用户可见中文以 [src/locales/zh-CN.json](../../src/locales/zh-CN.json) 为准；本表用于产品与文档对齐，若与本文件冲突应**改 JSON key 的取值**并同步更新本表描述。
- **Key 命名**：采用点分命名空间，例如 `page.assessmentList.title`、`nav.startAssessment`、`mock.assessment.a1.title`；页面级模块优先放在 `page.<routeSegment>.…`，全站共用放 `common.*`、`nav.*`、`breadcrumb.*`。
- **正式测评题干**：不放入 UI 字典，由后端或内容系统按语言返回（见 `page.assessmentAnswer.questionPlaceholder` 等仅占位说明）。

## 平台与导航

| 概念 | 统一用语 | 避免使用 |
|------|----------|----------|
| 平台整体 | 平台名称（立项后替换占位） | 系统、后台 |
| 首页 | 首页 | 主页、Main |
| 心理测评模块 | 心理测评 / 测评（列表页大标题可用「心理测评」） | 问卷系统、量表库 |
| 课程模块 | 课程 | 培训班、商品列表 |
| 顶栏 · 课程入口（对外） | **商城**（与「课程列表」同路由 `/courses`）或直接用「课程」——**选定一种全站统一** | 混用「商城」「课程」「小店」 |
| 顶栏 · 测评入口 | **心理测评** / **测评**（与下拉搭配） | 口语「测一测」单独作主导航 |
